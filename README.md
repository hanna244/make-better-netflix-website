## 네이밍 작성 규칙

### 커밋 메세지 

```
[구축]   ... 
[개선]   ...
[최적화]   ...
[업데이트]   ...
```

## 파일 탐색시 상대경로 규칙 
`./` 문구를 반드시 입력해준다. 

```html
<link href="./scss" />
<img src="./src" />
```
